Delay

 

The Delay feature in JCAT, allows the user to incorporate an element of time into a Model.  For example, if the Delay is set to 0, then the Event or Mechanism will happen immediately if it receives a causal signal.  If the Delay is set to 1, it will wait a timestep.  This allows users to model Events and Mechanisms which have a certain amount of lag to them.

 

For instance, if a commander orders an air strike against a factory, there will be some lag before the factory is destroyed due to mission Modelning and execution time.  

 

To add Delay:

 

1.  Select the Delay field with the mouse.

 

2.  Using the keyboard, enter a value.

 

3.  Then Select the OK button to accept the value and dismiss the window.
