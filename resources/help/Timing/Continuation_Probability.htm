Continuation Probability

 

If an Event or Mechanism gradually becomes less likely to occur as time progresses, this field should be set.  The value set for continuation multiplies the probability of an Event or Mechanism's every timestep until the probability gets closer to zero.  

 

For instance, if Continuation for an Event is set to 0.25, and the Event initially has the probability of 1.0, then after 1 timestep, it will have probability 0.25 (this is 1.0 * 0.25).  After another timestep, it will have the probability 0.0625 ( 1.0 * 0.25), and after another, the probability will be 0.015625 (1.0 * 0.25 * 0.25 * 0.25).
