Setting Scheduling

 

Step 5 of 6:

 

JCAT models temporally, so we must specify when events will occur.  If an event has no modeled causes and its leak probability is 0, its probability will remain at 0 for the whole simulation.  So, let’s suppose that Salvatore is caught wearing the wire at timestep 2, and we are 90% certain this event occurred.  Also, let us suppose that Tony takes Salvatore for the boat ride at timestep 5, and, again, we are 90% certain this event occurred.

 

To enter this information into the model, right click on “Caught wearing wire” and select “timing” from the popup menu again.  This time, we will deal with the bottom portion of the dialog box labeled “Scheduling.”  We said this will occur at time 2, so set the field labeled “Time” to 2, and set the field labeled “Probability” to 0.9.  Then click the “Schedule” button to add this to the scheduling list.

 

Repeat this for “Boat ride” at time 5 and probability 0.9.
