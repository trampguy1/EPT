AND / OR Expenditure

 

Every effect can be set to use it resources in either an AND or an OR manner. (An effect is represented rectangular box; lines represent mechanisms). If the effect is set for AND utilization, then all resources on mechanisms causing (i.e. lines pointing to) the effect are used together.  If the effect is set for OR utilization, then only the single best resource is selected. (i.e. a simple target/weapon pairing).
