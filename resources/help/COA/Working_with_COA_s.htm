Working with Courses of Action

The Course of Action dock window provides some other functionality relating to Courses of Action.  Aside from creating and applying a COA from the list, when right clicking on an item in the list you may also view a COA summary, a COA report, compare multiple COA's or delete the selected COA.

The View Summary allows modelers to enter extraneous information and documentation about the course of action.  The View Report option generates a textual representation of the course of action, listing all the tracked items and the values of the tracked parameters.  

The Compare tool allows for the side by side comparison of the probability outputs from multiple COA's for selected items.  To use the compare tool select compare and choose which COA's and items you would like to compare.  Then choose whether the color lines in the resulting graph should correspond to the model item or the COA.  At this time if you are unsure which selection to make, you may leave it as is and change it later.  Press the 'compare' button to generate the comparison graph.
