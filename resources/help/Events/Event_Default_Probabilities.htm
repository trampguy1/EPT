Default Probabilities of an Event

 

The Default Probabilities pane modifies the probabilities assigned to incoming and outgoing signals by default.  These probabilities, in turn, affect how this Event is influenced by various causes and inhibitors.

 

 

The probabilities specified here will be applied by default to any signals the Event receives or sends.  The probabilities for specific signals can be modified through the “Specified Probabilities” pane.
