Scheduling Events

 

One of the features that sets JCAT apart from other predictive analysis tools is the notion of temporal progression.  The user can specify the time at which each Event occurs and the probability with which it occurs at that time.

 

To set a time for an Event to occur, right click on the Event of interest and select “Timing” from the popup menu.  The following dialog box will appear:

  

 

Setting a time for an Event to occur is called Scheduling.  To Schedule the Event, look under the Scheduling portion of the dialog box.  Under the “Time” text field, enter the integer value of the timestep at which you would like the Event to occur.  

 

Next, under the “Probability” text field, enter in the probability that the Event will occur at this time.  When you have entered the desired time and probability, click the “Schedule” button.  The time and probability will then appear in the Scheduling table.  You can Schedule an Event for as many timesteps as you want.

 

To remove a timestep from Scheduling, simply select the row from the Scheduling table you wish to remove and click the “Delete” button.  To remove all Scheduling for the Event, click on the “Delete All” button.

 

For more information about Scheduling, see the Scheduling section in the Help file.
