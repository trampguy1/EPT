Creating a New Event

 

 

To create a new Event, we must first have an active Model open.  When you have opened the Model you wish to modify, determine where in the Event Layout Frame you wish to place the new Event.  Then click the Create Event button in the Model Toolbar, and click on the location in the Layout where you would like the new Event.

 

A square with rounded corners will appear.  The next step is to name the Event.  To edit information about an Event, right click on it, and select “Edit Event” from the popup menu.  The following dialog should appear:

 

 

To rename the Event, enter the new name in the “Event or Action Type” field.  Since an event usually involves some object or entity which performs it, the “Object to which it applies” field specifies which entity is performing the event.

 

For instance, if the Event is that Air Force Base X launches aircraft, then you would enter in “Launch aircraft” into the Event or Action type field, and you would enter “Air Force Base X” into the Object to which it applies field.

 

When you click the “Save” button in the Event Editor, your Event will be complete.  You can then connect this Event to others by adding Mechanisms.  Also, you can modify the

probabilities which govern how this Event is affected by incoming and outgoing signals.
