The Evidence Manager

 

Evidence is a way of introducing information from a detection or deduction into the model.  To specify evidence for a particular Event or Mechanism, right click on it and select “Evidence” from the popup menu.  The following dialog will appear:

 

 

 

There are two categories of Evidence supported by JCAT: Sensor Evidence and Absolute Evidence.  Both types have separate tabs in the dialog.  Sensor Evidence simulates data from a detector with two parameters: the False Alarm Rate and the Missed Detection Rate.  Absolute Evidence simulates data from some other outside source.
