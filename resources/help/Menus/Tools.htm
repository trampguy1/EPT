Tools Menu

 

The contains the following options:

 

 

Option

Description

 

Node Search

 

 

Searches the Model for Events with specified text or keywords

 

Spell Check Model

 

 

Checks the Events in the Model for misspelled words

Model Statistics

 

Provides data about the current Model including Event/Mechanism counts and file size

 

Graphical Documentation

 

Allows the user to export images and create PDF files to document any Probability Profile plots

 

Edit Signal Library

 

The Signal Library contains all the signals ever used in your model.  Even when deleted from the actual model signals remain in the library for future use.  This editor provides the user with the ability to view and delete signals from the model's library.

 

 

Security Options

 

Allows the user to change what port with which to connect to remote Collaboration sessions (password required- default password is primate)

 

User Preferences

 

Creates a dialog for the user to modify miscellaneous options
