Documentation Dialog

 

The allows the user to insert descriptions, comments, and key words into Models or Events.

 

To access the Documentation dialog for an Event, right click on the Event and select “Documentation” from the popup menu.  To access the Documentation dialog for a Model, select “Documentation” from the Tools Menu.

 

The Documentation dialog contains four tabs: Description, Reference Library, Comments, and Key Words.  The Description tab allows for a long description of the Model/Event and includes a spell check button to correct any misspellings in the text.   The Reference Library allows the user to list any references used in designing the Model/Event.  The Comments tab displays a list of any comments along with the date they were made.  The Key Words tab allows the user to associate any key words with the Model/Event.  Key words can later be used for a variety of purposes like Event searches.
