Mechanism Context Menu

 

When a user right clicks on a Mechanism within a Model, there are several options that may be of use to the user throughout the model building process.  Below you can find a list of the options and links to the sections which explain how each option is applied to the modeling process.

 

Option

Description

 

Plot

 

The plot option allows the modeler to plot selected Mechanisms in the Probability Profile.

 

Timing

 

This opens the Timing dialog.  See Timing Dialog section for further details.

 

Evidence

 

This opens the Evidence dialog.  See Evidence Manager section for further details.  

 

Set Font

 

This opens the Font Dialog for Font modification purposes.  

 

Arched Line

 

Arching the Mechanism line allows the modeler to curve the line around objects in the model.

 

Straight Line

 

Straightening the Mechanism line produces exact angles around objects in the model.  

 

Delete

 

This will allow the modeler to delete selected items.

 

Rename

 

Renaming allows the modeler to change the name of Mechanisms .

 

Resources

 

This opens the Resource Dialog for Resource editing.  See Resources for further details.
