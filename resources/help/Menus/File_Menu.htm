HELP

File Menu

 

 

The contains the following options:

 

 

Option

Description

 

New

 

Creates a new Model.

 

Open

 

Opens a saved Model.

Recent

 

A shortcut to open a recently used saved Model file

 

Import Siam XML

 

Allows the user to open and modify a Model created in Siam

 

Join Remote Session

 

Connects the user to a remotely started Collaboration session

 

Exit Remote Session

 

Closes the connection to the remotely started Collaboration session

 

Save

 

Saves the current Model file

 

Save As

 

Saves the current Model file to a new file

 

Close

 

Closes the current Model in the Model Layout Pane

 

Exit

 

Quits JCAT.
