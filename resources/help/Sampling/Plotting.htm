Plotting

 

Plotting is a feature in JCAT which allows users to observe how an Event behaves over time.  JCAT plots the probability of a selected event at each timestep in the simulation.  Several Events may be plotted together for comparison purposes.

 

Two profiles are displayed for each Event being plotted.  A solid line is the Predicted Probability Profile based on Scheduling and Timing.  A dotted line represents the Inferred Probability Profile based on observed Evidence.
