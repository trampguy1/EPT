User Preference Dialog

 

The User Preferences dialog allows users to modify miscellaneous options.

 

General Options

Option

Description

 

Simple probability mode

Enabling this mode will help users who are not familiar with the basic idea of probabilities by using word values instead of numbers throughout JCAT.  The name and value associations used by this feature may be modified by the user if desired.  To do this, the user must edit the file (maskedvalues.properties) in the directory <User_Home>/.JCAT.  Inside the file will be the current probability to word association which can be changed to any value so long as the format specified in the file is maintained.  Additional values may be added if desired to increase the granularity of user selectable values.

 

Enable Auto saving

 

With this function enabled, JCAT will automatically save the current Model to a temporary file for recovery purposes.  The user can specify the length of time between auto saves with the “Delay” field.

 

Recent files list size

 

Specifies the number of files listed in the Recent Files folder in the File Menu

 

Show grid lines

 

Toggles display of the square grid lines in the Model Layout Frame

 

Snap objects to grid

 

Toggles whether Events in the Model Layout Frame conform to sizes and locations fitted to the grid lines

 

Change node default font

 

This allows the user to modify the font used for new Events created in the Model Layout Frame

 

Highlight objects when editing

 

Toggles whether objects blink when they are being edited

Enable Debug Logging

Output more data to the JCAT log file.  This option helps in debugging errors but may reduce performance.

 

Event creation mode

 

This option changes the behavior when creating new Events.  Choose to either show the Event editor dialog, prompt for an Event name or take no additional action after creating the new Event.

Tool selection behavior

This option changes allows the user to change the behavior of the Plan Toolbar.  Choose to either reset the tool selection back to the 'select' tool each time an new Event or Mechanism is created or keep the tool being used selected until manually changed.

 

 

Color Scheme

The color scheme panel allows the user to define a custom color scheme for the current model.  Attributes can be created and assigned colors which will then be available for assignment to Events in the model.  Any later changes to the scheme will propagate through the model.
